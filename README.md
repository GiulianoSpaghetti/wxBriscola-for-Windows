# wxBriscola-for-Windows
[wxBriscola 0.5](https://github.com/numerunix/wxBriscola) per Visual Studio

# Come installare

winget install GiulioSorrentino.wxBriscola
Sull'ultimo windows 11 non passa i toast, perché non è firmato.
